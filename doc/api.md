# `smolarith` API

```{eval-rst}
.. role:: py(code)
   :language: python
```

`smolarith` does not export anything from the top-level; you must import the
module you want to use, perhaps with an alias, e.g.

```{doctest}
>>> import smolarith.mul as mul

```
